Data Science & Analytics Projects

Explore my professional journey through this portfolio, showcasing a selection of data projects that underscore my proficiency and versatility in the field. While respecting the confidentiality of client data, these works represent anonymized yet realistic scenarios, encompassing a spectrum of challenges from sentiment analysis in e-commerce to advanced predictive modeling.This portfolio is designed to be dynamic, evolving with ongoing additions that demonstrate my continuous growth and adaptability in the field of Data Science & Analytics.

Sentiment Analysis of Amazon Product Reviews Using Natural Language Processing

This project offers a comprehensive exploration of sentiment analysis techniques on Amazon product reviews, highlighting the efficacy of NLP in deciphering real-world e-commerce data. It adeptly combines VADER and RoBERTa models for a nuanced understanding of customer feedback, showcasing the transformative power of NLP in e-commerce analytics.

Regression Analysis for Transaction Forecasting Using CRISP-DM

This project employs the Cross-Industry Standard Process for Data Mining (CRISP-DM) framework to analyze and forecast transaction data over a three-year period, emphasizing data quality and actionable predictions. After evaluating five different models, the Random Forest regression emerged as the top-performing model, ensuring accurate and reliable transaction forecasting.

Analyzing U.S. Economic Trends Using FRED API

This project showcases the power of the FRED (Federal Reserve Economic Data) API by using it to retrieve and analyze data related to the S&P 500 index, unemployment rates, and labor force participation rates at the state level.

Data Cleaning and Transformation in SQL: Nashville Housing

In this project, I demonstrate my proficiency in data cleaning and transformation using SQL. I delve into the process of standardizing date formats, populating missing property addresses, breaking down address data into individual columns, and more.

Data Analysis in SQL: COVID-19 Vaccination Progress and Population Impact

In this SQL project, I analyze COVID-19 vaccination progress and its impact on populations. Using SQL queries, I extract and examine critical COVID-19 data, including total cases, total deaths, vaccination rates, and population statistics.

Exploring Correlations in Movie Data: Votes and Budget vs. Gross Revenue

In this exploratory data analysis (EDA), we delve into the world of movies by analyzing the correlations between key variables. We uncover that votes and budget exhibit the strongest correlations with gross revenue, dispelling prior assumptions about the influence of the production company.
